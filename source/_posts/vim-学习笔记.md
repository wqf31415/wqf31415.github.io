### vim 是啥?
Vim 是一个高度可配置的文本编辑器，能够高效的创建、修改多种类型文本。在多数 UNIX 系统和 Apple OS X 中集成了 vi。(vi 是 vim 的子集，vim 具有 vi 的所有功能，且拥有更多的实用特性)。
> 官网: <https://www.vim.org/> 

<!-- more -->

### 为啥要用 vim ?

vim 是 linux 系统中常用的文本编辑器，在运维 linux 服务器时经常使用，比如用来修改配置文件。
