### 概述

这篇文件介绍了 Rust 中最核心，也是最难理解的两个概念：所有权和生命周期，想学好 Rust 就必须充分的理解这两个概念。所有权是 Rust 最特别的一个理念，在其他编程中没有所有权的概念，因此从其他编程语言转来学 Rust 时会难以理解。生命周期在大多数编程语言中都有，但都是隐式的，在 Rust 中可以显式的指定生命周期，这也是 Rust 所独有的。
